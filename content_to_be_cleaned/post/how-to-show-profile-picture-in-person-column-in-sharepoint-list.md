## Introduction {#toc-hId--494643285}

 

Currently, Microsoft released a new update to the show profile picture
of the user column in the SharePoint list/library. So in this article,
we will see how to use this feature in our list or library. for more
details refer to
[this](https://techcommunity.microsoft.com/t5/microsoft-365-blog/what-s-new-for-microsoft-lists-microsoft-ignite-2021-mar-2-4/ba-p/2176242 "SharePoint List Updates").

 

 

## How to use it? {#toc-hId-1992869548}

 

1\. Log in to any SharePoint Modern Site.

2\. Move to the modern list/library

3\. Create a Person column in List/Library as below,

 

 

[![22.png](https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271652i62B29CEB2AF5FA27/image-size/large?v=v2&px=999 "22.png"){.lia-media-image
role="button"
li-image-url="https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271652i62B29CEB2AF5FA27?v=v2"
li-image-display-id="'271652i62B29CEB2AF5FA27'"
li-message-uid="'2265060'" li-messages-message-image="true"
li-bindable="" tabindex="0" li-bypass-lightbox-when-linked="true"
li-use-hover-links="false"}]{.lia-inline-image-display-wrapper
.lia-image-align-center image-alt="22.png" style="width: 743px;"}

 

4\. Turn on to show profile photo to see profile URL.

 

[![1.png](https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271653i9FB9527725FB3E1E/image-size/large?v=v2&px=999 "1.png"){.lia-media-image
role="button"
li-image-url="https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271653i9FB9527725FB3E1E?v=v2"
li-image-display-id="'271653i9FB9527725FB3E1E'"
li-message-uid="'2265060'" li-messages-message-image="true"
li-bindable="" tabindex="0" li-bypass-lightbox-when-linked="true"
li-use-hover-links="false"}]{.lia-inline-image-display-wrapper
.lia-image-align-center image-alt="1.png" style="width: 227px;"}

 

 

4\. Then create a list item and check it in the list/library view.

 

## [![3.png](https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271654iDB07C141434B3453/image-size/large?v=v2&px=999 "3.png"){.lia-media-image role="button" li-image-url="https://techcommunity.microsoft.com/t5/image/serverpage/image-id/271654iDB07C141434B3453?v=v2" li-image-display-id="'271654iDB07C141434B3453'" li-message-uid="'2265060'" li-messages-message-image="true" li-bindable="" tabindex="0" li-bypass-lightbox-when-linked="true" li-use-hover-links="false"}]{.lia-inline-image-display-wrapper .lia-image-align-center image-alt="3.png" style="width: 999px;"} {#toc-hId-185415085}

In this article, we have seen how to show the profile picture in the
person column.
